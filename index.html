Looking for free web hosting then you are on the right place.In this blog you will learn how to host website on github for free with full guidelines step by step.There are many web hostings that provide free services to host website but they have some limitations

Free Web Host Providers

There are many free web hosting providers that let you host your static website for free but there is some limitations

  1. You can host 2 or 3 websites for free
  2. Sometimes website load time increases because of high traffic on same server and priority for paid hosted websites.
  3. Brand name display on your sites.

Why To Choose Github for web hosting ?

  1. Its completely free
  2. No brand name display on your site
  3. Faster reload time
  4. Easy to deploy
  5. You can host many websites for free

Host Website on Github.

Coder Dreamer will guide you step by step how to host website on gitub

.

Create Your Github Account.

First You have to create your github account.If you don't have github account, here is the link Github Sign up and create your github account

Create Github Repository

After Successfully creating your github account,create your github repository by clicking on create repository.

give any name to your github repository e.g testing

make sure that your github repository are public.

Upload Files on Github Repository

Click on upload an existing files

Select your web(coding files) files and upload on repository by drag and drop or you can just upload files without drag and drop by click on choose your files.

Commit Changes

After uploading your webfiles, name your commit file or just leave empty and click on commit changes.

Go to Settings

Click on the setting menu

Click on Pages

Choose the main branch from drop down and click on the save button

Click on Live Url

Now,wait for 2 to 3 minutes,github hosting your site.After 3 minutes refresh the page you will see the message on the top your site is live at x,y,z like in the image.
